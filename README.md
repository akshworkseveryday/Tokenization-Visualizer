**Tokenization Visualizer
**
**Overview
**
This project is an interactive web application created to explore and understand different tokenization strategies used in Natural Language Processing and modern Large Language Models.

As a student learning about Artificial Intelligence, language models, and intelligent agents, I built this project to apply theoretical knowledge in a practical setting. The goal is to better understand how raw text is converted into tokens before being processed by AI systems.


**Objectives
**
The primary objective of this project is to strengthen practical understanding of text preprocessing techniques in NLP. It focuses on observing how different tokenization methods segment the same input text and how those differences influence token count and representation.


Supported Tokenization Methods

The application currently supports the following tokenization methods.

Word Tokenization  
Character Tokenization  
Byte Pair Encoding using GPT 2  
WordPiece using BERT  
Unigram Language Model using T5 and SentencePiece  

These approaches represent both classical and modern tokenization techniques widely used in transformer based architectures.


**Features
**
The application provides an interactive text input area where users can enter custom sentences. 
It allows selection between different tokenization methods and displays the total number of tokens and characters. 
The tokenized output is visualized using alternating pastel colors to clearly highlight token boundaries. 
There is also an option to display the token IDs generated by each tokenizer.

**Tech Stack
**
The project is built using **Python** and **Streamlit** for the web interface.
It integrates **HuggingFace Transformers** and **SentencePiece** to access pretrained tokenizers commonly used in real world NLP systems.


**How to Run Locally
**
Install the required dependencies:
pip install -r requirements.txt
Run the application using:
streamlit run app.py
